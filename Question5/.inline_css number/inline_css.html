ADVANTAGES OF CSS

CSS saves time
We can write CSS once and then reuse the same sheet in multiple HTML pages. We can define a style for each HTML element and apply it to as many webpages as we want
Pages load faaster
if we are using CSS, we do not need to write HTML tag attributes everytime. Just write one CSS rule of a tag an apply it all the occurances of that tag. so less code means faster download all times
Easy maintenance
To make Global change, simply change the style and all the elements in all the webpages will be updated automatically
Multiple Device Compantability
Stylesheets allow content to be optimised for more than one type of device By using the same HTML document,different versions of a website can be presented for handheld devices such as PDAs and cell phones or for printing

We can use CSS in 3 ways

  • External StyleSheets
  • Internal StyleSheets
  • Inline StyleSheets
